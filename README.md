# Google Play Store Clustering
Proyek ini bertujuan untuk melakukan pengelompokan (clustering) aplikasi Google Play Store
berdasarkan karakteristik seperti rating, jumlah review, jumlah install, dan ukuran aplikasi.
Clustering dilakukan untuk mengetahui pola dan segmentasi aplikasi di Google Play Store.
Studi kasus yang digunakan adalah dataset Google Play Store yang berisi informasi aplikasi
seperti rating, jumlah ulasan (reviews), jumlah instalasi (installs), dan ukuran aplikasi (size).
Dataset diperoleh dari:
- Sumber: Kaggle â€“ Google Play Store Apps
- Format: CSV
- Lokasi file: folder `dataset/`
Tahapan yang dilakukan dalam proyek ini meliputi:
1. Data Acquisition
2. Data Preprocessing (cleaning dan transformasi data)
3. Exploratory Data Analysis (EDA)
4. Penerapan metode clustering menggunakan K-Means
5. Evaluasi hasil clustering
6. Interpretasi hasil
Metode data mining yang digunakan adalah K-Means Clustering.
Proses scaling data dilakukan menggunakan StandardScaler sebelum clustering.
Output dari proyek ini berupa hasil clustering aplikasi Google Play Store
yang ditampilkan dalam bentuk visualisasi dan analisis cluster.
